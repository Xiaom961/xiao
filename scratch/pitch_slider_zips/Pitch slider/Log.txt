How can i change the pitch of the output audio when im using kokor TTS?


I have successfully implemented real-time Pitch Control for Kokoro TTS in the output audio!

Here is exactly what I've done:

Enabled the UI Pitch Slider for Kokoro TTS:

In 

speech.ts
, I updated the supportsPitch capability checker to return true when the active provider is kokoro-local.
This immediately triggers the UI to render the native Pitch Slider under Settings ➔ Intelligence ➔ Modules ➔ Speech when Kokoro is selected.
Hooked up Output Audio Pitch shifting (via Web Audio API playbackRate):

In 

Stage.vue
, in the playFunction that drives character voice playback, I added a targeted dynamic scaler for kokoro-local.
When playing speech generated by Kokoro, it reads the current value of the pitch slider and maps it to the AudioBufferSourceNode's playbackRate.
A pitch setting of 0 plays back at the normal 1.0 rate. Adjusting the slider will scale the rate smoothly between 0.5 (very deep/slow) up to 1.5 (very light/fast/higher-pitched), allowing you to fine-tune the exact tone you want in real time!
Validated Codebase & Formatted Code:

Ran a monorepo typecheck to ensure all changes are type-safe.
Formatted both updated files using the project’s strict ESLint style guidelines to ensure clean integration with the main repository.
🌟 How to use it now:
Go to Settings ➔ Intelligence ➔ Modules ➔ Speech.
Make sure Kokoro TTS is active.
You will now see the Pitch slider! Drag it to the right (positive values) to instantly raise the pitch of the voice, or to the left (negative values) to make it deeper.
Try playing a test message or talking to your character—the character's speech pitch will adjust instantly in real time!
